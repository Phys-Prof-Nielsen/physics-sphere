Problem D3.5

In open-heart surgery, a much smaller amount of energy will defibrillate the heart.

  1. What voltage is applied to the \(8.00~\mu\)F capacitor of a heart defibrillator that stores 40.0 J of energy

  2. Find the amount of the stored charge.

This problem is a slightly modified version from OpenStax. Access for free here

Show code cell source Hide code cell source 
%reset -f

import sympy as sym

C = 8.00E-6
U = 40.0

# U = 0.5*C*V^2 
V = sym.Symbol('V')
eq = U - 0.5*C*V**2
sol = sym.solve(eq,V)

print('Voltage applied to capacitor is '+str(sol[1])+' V.')
Show code cell output Hide code cell output 
Voltage applied to capacitor is 3162.27766016838 V.
